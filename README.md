# 🔎 Noxis Frontend

Frontend interface for the Noxis file indexing and search system.

---

## 🚀 Overview

Noxis Frontend is the visual layer of the Noxis ecosystem, built to provide a fast, modern, and intuitive experience for searching, previewing, and accessing indexed files.

The interface was designed with a strong focus on performance, usability, and a clean visual style inspired by modern productivity tools.

---

## 🧠 Features

- 🔎 Instant file search by name, path, or extension  
- 📄 File preview support  
- ⬇️ File download access  
- 📊 Search performance indicators  
- 🧭 Directory navigation with breadcrumb support  
- 🌙 Modern dark interface  
- 📱 Responsive design for desktop and mobile  

---

## 🛠 Tech Stack

- Vue.js  
- Vite  
- Tailwind CSS  
- Lucide Icons  
- FastAPI integration  

---

## 🏗 Architecture

- **Frontend (Vue.js)** → handles the user interface and interactions  
- **Build tool (Vite)** → provides fast development and build performance  
- **Styling (Tailwind CSS)** → supports a modern and responsive UI  
- **Icons (Lucide Icons)** → improves interface clarity and usability  
- **Backend integration (FastAPI)** → connects the interface to the Noxis search engine
