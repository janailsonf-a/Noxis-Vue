# 🚀 Noxis Frontend

Interface moderna e performática para o sistema **Noxis**, um indexador corporativo de arquivos projetado para buscar milhões de registros com alta velocidade.

---

## 🔎 Sobre o projeto

O **Noxis Frontend** é a camada visual do sistema de indexação de arquivos.

Ele foi desenvolvido com foco em:

- ⚡ Performance extrema
- 🎯 Experiência do usuário (UX)
- 🎨 Design moderno (inspirado em Notion / Google Drive)
- 🔍 Busca rápida com destaque de resultados
- 📁 Navegação intuitiva (grid / lista)

---

## 🧠 Funcionalidades

- 🔍 Busca instantânea por nome, caminho ou extensão
- 📄 Preview de arquivos (PDF, imagens, vídeos)
- ⬇️ Download de arquivos
- 📊 Indicadores de performance (tempo de busca, resultados)
- 🧭 Navegação por diretórios (breadcrumb)
- 🌙 Interface dark moderna
- 📱 Responsivo (desktop e mobile)

---

## 🛠️ Tecnologias

- Vue.js (Vite)
- Tailwind CSS
- Lucide Icons
- Integração com API FastAPI
